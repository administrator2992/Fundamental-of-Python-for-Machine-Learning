# Fundamental of Python for Machine Learning
## Dasar-dasar Pemrograman Python Untuk Machine Learning dan Kecerdasan Buatan

Dalam github ini berisi tentang rangkuman dari buku **FUNDAMENTAL OF PYTHON FOR MACHINE LEARNING** ditulis oleh Bapak Teguh Wahyono yang merupakan dosen Fakultas Teknologi Informasi UKSW Salatiga. Alumni S1 Teknik Informatika UKDW Yogyakarta, S2 Ilmu Komputer UGM dan saat ini menempuh Program Doctor of Computer Science di Bina Nusantara University Jakarta. Hobi menulis buku bidang TI sejak tahun 2002, dan dalam catatan penulis ini adalah buku ke 57 yang pernah ditulisnya pada beberapa penerbit nasional.

Struktur pembahasannya meliputi 3 bagian besar yaitu (1) Konsep *Machine Learning* dan Kecerdasan Buatan (2) Dasar-Dasar Pemrograman Python untuk *Machine Learning* serta (3) Contoh Penerapan *Machine Learning* maupun *Unsupervised Learning.* Beberapa studi kasus dibahas secara lengkap dari pemahaman algoritma, megolah dataset samapi pada *training* dan *testing* serta visualisasi hasil dari model *machine learning* yang dikembangkan.
